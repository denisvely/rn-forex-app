## Project Name & Pitch

React Native Forex App

## Technologies used

1. React Native
2. Expo

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Start Server:

`npm start`

To Visit App:

Scan the QR code with Expo App on your phone.
